### 📌 Por qué esta ruta

- **Python → Nix/Bash**: refuerza lógica y me da soltura total en la terminal de Linux/NixOS.
- **HTML/CSS → JavaScript**: consolida el front-end antes de saltar a lenguajes de bajo nivel.
- **C**: entender memoria y punteros hace que Rust se sienta natural después.
- **Go**: puente amigable entre lenguajes de alto nivel y sistemas, con concurrencia simple.
- **Rust 🦀**: meta final — seguridad de memoria sin garbage collector, ideal tras este recorrido.
